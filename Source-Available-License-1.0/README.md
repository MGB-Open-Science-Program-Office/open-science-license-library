# **MGB Source Available License 1.0**

MGB recommends a specific license for restrictive source available distribution, the MGB Source Available License (MGBSA 1.0).

This directory contains the text of the [license](https://github.com/MGB-Open-Science-Program-Office/open-science-license-library/blob/main/Source-Available-License-1.0/mgb-source-available-license-1.0.md) and [best practices documentation](https://github.com/MGB-Open-Science-Program-Office/open-science-license-library/blob/main/Source-Available-License-1.0/best-practices-source-available-license-1.0.md) on how to use it.

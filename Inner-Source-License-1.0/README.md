# **MGB Inner Source License 1.0**

MGB recommends a specific license for inner source development restricted to distribtion within the system, the MGB Inner Source License (MGBISL 1.0).

This directory contains the text of the [license](https://github.com/MGB-Open-Science-Program-Office/open-science-license-library/blob/main/Inner-Source-License-1.0/mgb-inner-source-license-1.0.md) and [best practices documentation](https://github.com/MGB-Open-Science-Program-Office/open-science-license-library/blob/main/Inner-Source-License-1.0/best-practices-inner-source-license-1.0.md) on how to use it.

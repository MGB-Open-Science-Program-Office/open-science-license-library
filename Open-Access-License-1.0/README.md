# **MGB Open Access License 1.0**

MGB recommends a specific license for open access distribution, the MGB Open Access License (MGBOA 1.0).

This directory contains the text of the [license](https://github.com/MGB-Open-Science-Program-Office/open-science-license-library/blob/main/Open-Access-License-1.0/mgb-open-access-license-1.0.md) and [best practices documentation](https://github.com/MGB-Open-Science-Program-Office/open-science-license-library/blob/main/Open-Access-License-1.0/best-practices-open-access-license-1.0.md) on how to use it.
